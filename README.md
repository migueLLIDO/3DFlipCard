# 📇 3D Flip Card - Efeito de Rotação Interativo

Este projeto apresenta um card de perfil elegante e moderno, utilizando transformações 3D avançadas no CSS para criar um efeito de "flip" (giro) quando o usuário passa o mouse. A interface é limpa, com uma estética refinada que valoriza a experiência do usuário.

## 🌟 Descrição

O componente consiste em um card que possui duas faces:
- **Face Frontal:** Exibe uma imagem de alta qualidade com bordas arredondadas e uma sombra suave.
- **Face Traseira:** Revela informações detalhadas do perfil, incluindo nome, cargo, uma foto de perfil circular e ícones de redes sociais com efeitos de hover vibrantes.

O efeito de transição é suave, utilizando a propriedade `perspective` e `rotateY` para simular profundidade e realismo.

## 🛠️ Funcionalidades Técnicas

- **Transformações 3D:** Uso de `perspective`, `rotateY` e `preserve-3d` para o efeito de giro.
- **Transições Suaves:** Animações controladas via `cubic-bezier` para um movimento natural.
- **Layout Moderno:** Utilização de Flexbox para alinhamento preciso dos elementos internos.
- **Design Responsivo:** O card se adapta bem a diferentes tamanhos de tela.
- **Efeitos de Hover:** Botões de redes sociais com gradientes dinâmicos e inversão de cores.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica do componente.
- **CSS3:** Estilização avançada, gradientes, sombras e animações 3D.
- **Font Awesome:** Ícones vetoriais para redes sociais.
- **Google Fonts:** Tipografia moderna (Poppins).

## 📂 Como Usar

1. Clone o repositório ou baixe os arquivos.
2. Certifique-se de que a estrutura de pastas (`css/`, `img/`) está mantida.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
